# Roadmap

## Completed MVP Features

- Admin authentication and role-based authorization.
- Admin dashboard.
- Client management.
- Project management.
- Timesheet entry management.
- Timesheet approval workflow.
- Invoice creation from approved billable timesheets.
- PDF invoice export.
- Company profile management.
- Read-only client dashboard.
- Read-only client timesheets page.
- Read-only client invoices page.
- Read-only client company profile page.
- Role-based navigation.
- Docker Compose support for PostgreSQL and pgAdmin.

## Next Features

- Refine client portal styling and empty states.
- Add date and status filters to client-facing tables.
- Add invoice detail views.
- Add timesheet detail views.
- Improve validation and error handling.
- Add automated tests for role-based access and client scoping.
- Add production SMTP configuration.
- Add deployment configuration for a hosted environment.

## Future Commercial/SaaS Features

- Multi-tenant company accounts.
- Subscription billing and plan management.
- Branded client portals.
- Advanced dashboards and analytics.
- Grafana reporting integration.
- Audit logs for sensitive actions.
- Backup and restore workflows.
- Role and permission administration.
- Secure production hosting and monitoring.
